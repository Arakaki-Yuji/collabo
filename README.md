# Collabo

オープンコラボレーションコミュニティ Collabo

人々がつながり合って助け合い、様々の問題をともに解決する社会を目指す

# Development Environment

## Compile ClojureScript

```
# clojurescriptを一回ビルドする
$ lein cljsbuild once

# clojurescriptをwatchして更新があるたびにビルドする
$ lein cljsbuild auto
```
